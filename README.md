# BodyQ - your customised fitness App

## 1. DASHBOARD

### how to use it

Install the document

- add a .env file in your backend folder and add in it the following:

```env
PORT = 5000 # any port number will do
URL = "your_database_url" # mongoDB Atlas for collaboration
```

- installations for frontend
  - npm install
  - npm install axios
- installations for backend
  - npm install
  - npm install express
  - npm install cors
  - npm install mongoose
  - npm install mongodb
  - npm install nodemon

to run your project run at the same time both folders (two terminals)

```terminal
cd frontend
npm run dev
```

```terminal
cd backend
npm start
```

## 2. APP

- cd mobile-frontend
- npx expo start
